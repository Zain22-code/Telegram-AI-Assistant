# Telegram-AI-Assistant

## Overview
An AI-powered Telegram bot built with n8n and Google Gemini to receive messages and generate intelligent responses.

## Technologies
- n8n
- Telegram Bot API
- Google Gemini API

## Features
- Receive Telegram messages
- AI-generated responses
- Automated chatbot workflow
